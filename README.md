# Random Moving Squares (Pygame)

This project is a simple, colorful simulation where squares bounce around the screen, collide, and sometimes multiply. It is designed for fun and as a learning tool for Python and Pygame.

## What You Need
- A computer (Windows, Mac, or Linux)
- Python 3.8 or newer
- Internet connection (for installing Python and Pygame)

## How to Set Up (Step by Step)

### 1. Install Python
If you don't have Python:
- Go to https://www.python.org/downloads/
- Click the big "Download Python" button.
- Open the downloaded file and follow the instructions to install.

### 2. Open a Terminal or Command Prompt
- **Windows:** Press the Windows key, type `cmd`, and press Enter.
- **Mac:** Press Command + Space, type `Terminal`, and press Enter.
- **Linux:** Press Ctrl + Alt + T, or find "Terminal" in your applications.

### 3. Install Pygame
Type this command and press Enter:
```
pip install pygame
```
If you see an error, try:
```
python -m pip install pygame
```

### 4. Download the Project Files
- Download or copy the entire `lab8-pygame` folder to your computer.
- Make sure the file `main.py` is inside the folder.

### 5. Run the Program
In your terminal, use the `cd` command to go into the project folder. For example:
```
cd Desktop/EPITA\ (Linux)/AI\ for\ Software\ Engineering/Github/lab8-pygame
```
Then run:
```
python main.py
```
If that doesn't work, try:
```
python3 main.py
```

## What You Will See
- A window will open with colorful squares bouncing around.
- The number of squares and their speed may change as they collide.
- Some squares may "pulse" (get brighter) when they hit something.
- The number of squares is shown at the top left.

## How to Quit
- Click the "X" at the top right of the window.
- Or, press `Ctrl + C` in the terminal.

## Troubleshooting
- If you see an error about `pygame` not found, repeat step 3.
- If the window doesn't open, make sure you are in the correct folder and try again.
- If you get a sound error, you can ignore it or turn off your speakers.
